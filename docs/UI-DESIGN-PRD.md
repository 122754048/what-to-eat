## 七、设计规范

### 色彩系统

| 角色 | 颜色 | 用途 |
|------|------|------|
| 背景 | `#FFFFFF` | 主背景 |
| 主色 | `#7CB97D` | 标签、小面积点缀 |
| 强调色 | `#F4A261` | CTA 按钮 |
| 主文字 | `#1D1D1F` | 标题、正文 |
| 次文字 | `#86868B` | 辅助说明 |
| 分割线 | `#E5E5EA` | 分隔线 |
| 卡片背景 | `#F5F5F7` | 卡片底色 |

### 间距系统（8pt 网格）

| 元素 | 数值 |
|------|------|
| 屏幕边距 | 24px |
| 元素间距 | 8 / 16 / 24 / 32px |
| 卡片圆角 | 20px |
| 按钮圆角 | 16px |
| 标签圆角 | 20px |

### 字号系统

| 用途 | 字号 | 字重 |
|------|------|------|
| 大标题 | 34px | Bold |
| 页面标题 | 28px | Semibold |
| 卡片标题 | 24px | Semibold |
| 正文 | 17px | Regular |
| 辅助文字 | 15px | Regular |
| 标签文字 | 12px | Medium |
| Tab 文字 | 10px | Medium |

### 动效规范

| 动效 | 参数 |
|------|------|
| 页面切换 | Spring, damping: 0.8 |
| 点击反馈 | scale(0.96) → scale(1.0) |
| 过渡时长 | 0.2-0.3s |
| 卡片拖拽 | 实时旋转 + 透明度变化 |

### 图标规范

- 使用 SF Symbols 线性图标
- 禁止使用 emoji 作为图标
- 尺寸：24px（标准）、20px（小）
