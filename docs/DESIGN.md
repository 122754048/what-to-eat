# 设计规范 - 「吃什么」(What to Eat)

> **版本**: v2.0
> **更新日期**: 2026-03-27
> **状态**: 正式版

---

## 1. 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#8B9A6D` | 鼠尾草绿，用于标签、Tab选中、小面积点缀 |
| 辅色 | `#F4A261` | 暖橙色，用于CTA按钮、强调元素 |
| 背景色 | `#FFFFFF` | 纯白，页面主背景 |
| 主文字 | `#1A1A1A` | 深灰黑，标题、正文 |
| 次文字 | `#86868B` | 中灰，辅助说明、占位符 |
| 分割线 | `#E5E5EA` | 浅灰，分隔线、边框 |
| 卡片背景 | `#F5F5F7` | 极淡灰，卡片内背景 |
| 标注色 | `#CCCCCC` | 灰色，未选中状态 |

**使用原则**：
- 主色仅用于标签、小面积点缀，禁止大面积使用
- 辅色仅用于CTA按钮，禁止滥用
- 背景色为纯白，保持干净通透

## 2. 字体规范

| 用途 | 字号 | 字重 | 示例 |
|------|------|------|------|
| 大标题 | 34pt | Bold | 页面主标题 |
| 页面标题 | 28pt | Bold | 区块标题 |

| Tab文字 | 10pt | Medium | 底部Tab |

**字体栈**：`SF Pro Display, SF Pro Text, -apple-system, BlinkMacSystemFont, sans-serif`

## 5. 动效规范

| 动效 | 参数 |
|------|------|
| 页面切换 | Spring, damping: 0.8 |
| 点击反馈 | `scale(0.96)` → `scale(1.0)` |
| 过渡时长 | 0.2-0.3s |
| 阴影动画 | `0 4px 24px rgba(0,0,0,0.08)` |

---

## 6. 图标规范

- **图标库**：SF Symbols
- **图标尺寸**：24px（标准）、20px（小）
- **图标样式**：线性图标，描边 1.5px
- **禁止**：使用 emoji 作为功能图标
