## 二、测试用例

### TC-001: 首页滑动卡片 - 向右滑动（喜欢）

- **用例编号：** TC-001
- **用例标题：** 首页滑动卡片 - 向右滑动（喜欢）
- **前置条件：** App 启动完成，Mock 数据加载成功，首页可见
- **优先级：** P0
- **测试步骤：**
  1. 启动 App，等待首页加载完成
  2. 确认屏幕中央显示菜品卡片
  3. 从卡片中间偏右位置开始，向右水平拖动卡片
  4. 拖动距离超过阈值（约100pt），卡片甩出
- **预期结果：**
  1. 卡片跟随手指拖动，有旋转效果
  2. 右侧显示绿色心形 Like 指示器，透明度渐变
  3. 松手后卡片向右滑出屏幕
  4. 下一张卡片自动出现
